   答：跨平台开发，一套代码多端运行（H5+小程序）

2. **JWT 认证的优势？**  
   答：无状态、可扩展、支持跨域

3. **前后端如何交互？**  
   答：RESTful API + JSON，axios 封装请求

4. **数据库表设计考虑？**  
   答：用户表、内容表、订单表等，外键关联
